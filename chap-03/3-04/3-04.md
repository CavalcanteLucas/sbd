# 3.04

Considere o banco de dados de seguro da Figura 3.17, onde as chaves primárias estão sublinhadas. Construa as seguintes consultas SQL para esse banco de dados relacional.

<pre>
    pessoa(<u>id_motorista</u>, nome, endereço)
    carro(<u>renavam</u>, modelo, ano)
    acidente(<u>num_sinistro</u>, ano, local)
    possui(<u>id_motorista</u>, <u>renavam</u>)
    participou(<u>num_sinistro</u>, <u>renavam</u>, id_motorista, valor_dano)
</pre>

**Figura 3.17** - Banco de dados de seguro.

**a)** Ache o número total de pessoas que possuíram carros e estiveram envolvidas em acidentes em 2017.

**b)** Exclua todos os carros do ano de 2010 pertencentes à pessoa cuja ID é `12345`.

## Resposta

Para preparar a tabela o banco de dados de seguro da Figura 3.17, utilize as instruções no arquivo `3-04-0.sql`.

**a)**

Aqui, precisamos considerar pessoas com vários acidentes apenas uma vez. Por isso, precisamos da cláusula `DISTINCT`.

Além disso, precisamos considerar o dono do carro envolvido no acidente, e não o condutor que participou no acidente. Por isso, precisamos obter o `renavam` do carro envolvido no acidente para encontrar seu dono.

Juntando tudo, temos:

```
select count(distinct po.id_motorista)
from seguro.possui po
join seguro.participou pa
  on po.renavam = pa.renavam
join seguro.acidente ac
  on ac.num_sinistro = pa.num_sinistro
where ac.ano = 2017;
```

| count |
| ----- |
| 3     |

**b)**

Antes de modificar o banco, lembre-se de verificar o estado estado atual das tabelas possivelmente afetadas. Note que, por causa das restrições de chave estrangeira `carro_fk` para `seguro.carro` em `seguro.possui` e `seguro.participou`, caso um registro de carro seja deletado os registros de pessoas que o possuem e os registros de acientes envolvendo esse carro também serão deletados.

```
select count(*) from seguro.carro;
select count(*) from seguro.possui;
select count(*) from seguro.participou;
```

Precisamos obter o ID dos carros que o motorista com o id `12345` possui. Em seguida, deletamos todos os carros de 2010 inclusos nesse conjunto.
```
delete from seguro.carro ca
where ca.ano = '2010' and ca.renavam in (
  select po.renavam
  from seguro.possui po
  where po.id_motorista = '12345'
);
```
