# 08-03

Uma companhia de aluguel de carros mantém em um banco de dados todos os veículos em sua frota atual. Para todos eles, ela inclui Renavam do veículo, placa, fabricante, modelo, data de compra e cor. Dados especiais são incluídos para certos tipos de veículos:

- Caminhões: capacidade de carga.
- Carros esportivos: HP, requisito de idade do cliente.
- Vans: números de passageiros.
- Veículos *off-road*: distância do solo, transmissão (tração nas quatro ou duas rodas).

Construa uma definição de esquema SQL para esse banco de dados. Use a herança onde for apropriado.
