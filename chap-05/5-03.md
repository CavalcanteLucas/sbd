# 5.3

Suponha que queiramos achar todos os cursos que devem ser realizados antes de determinado curso. Isso significa achar não apenas os pré-requisitos desse curso, mas os pré-requisitos dos pré-requisitos e assim por diante. Usando JDBC, escreva um programa Java completo que:

- Apanhe um valor de `id_curso` do teclado.
- Ache os pré-requisitos desse curso usando uma consulta SQL submetida via JDBC.
- Para cada curso retornado, ache seus pré-requisitos e continue esse processo iterativamente, até que nenhum curso de pré-requisito seja achado.
- Informe o resultado.

Nesse exercício, não use uma consulta SQL recursiva, mas sim a abordagem iterativa descrita anteriormente. Uma solução bem desenvolvida será resistente ao caso de erro em que uma universidade cria, acidentalmente, um ciclo de pré-requisitos (por exemplo, o curso A é um pré-requisito para o curso B, o curso B é um pré-requisito para o curso C e o curso C é um pré-requisito para o curso A).
