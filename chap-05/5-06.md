**Figura 5.21** - Banco de dados de agência bancária.

Vamos definir uma *view* $agência\_cliente$ da seguinte forma:

```
create view agência_cliente as
  select nome_agência, nome_cliente
  from depositante, conta
  where depositante.num_conta = conta.num_conta
```

Suponha que a *view* seja *materializada*, ou seja, a *view* é calculada e armazenada. Escreva *triggers* para *manter* a *view*, isto é, mantê-la atualizada nas inserções e exclusões de $depositante$ ou $conta$. Não se preocupe com as exclusões ou atualizações. Observe que, para simplificar, não exigimos a eliminação de duplicatas.
